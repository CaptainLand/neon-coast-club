# Neon Coast Club

Neon Coast Club 是一个使用 Godot 4 与 GDScript 开发的第一人称 3D 互动舞厅原型。项目融合 Y2K / 蒸汽波海岸场景、可交互道具、物理玩法、昼夜系统、多人实验功能，以及可从街机进入的 4K 下落式音游。

## 核心内容

- 第一人称移动、冲刺、跳跃、自由飞行和射线交互
- 程序化舞厅、海岸、夜景城市、灯光和动态昼夜变化
- 街机、CRT、扭蛋机、收藏品、物品栏和可投掷物理道具
- FPS 武器、换弹、瞄准、命中特效和基础 HUD
- Godot 在线大厅、账号验证、双人房间与 1v1 回合制对战
- 模块化场景、脚本、Shader 和第三方素材许可记录

## 1v1 对战模式

项目内置离线训练和在线 1v1 两种对战方式。玩家可在启动界面进入在线大厅，注册或登录账号，创建或加入一个双人房间；两名玩家全部准备后，服务器自动开始比赛。

### 对战流程

1. 连接统一大厅并完成账号登录
2. 创建房间或从大厅列表加入房间
3. 双方点击准备，房间状态切换为“对局中”
4. 每回合开始前冻结 3 秒，随后进行 90 秒对战
5. 玩家被击败或回合时间结束后显示胜负结果、累计比分并自动重置下一回合

### 战斗与同步

- 提供突击步枪、冲锋枪、霰弹枪、左轮手枪和战术刀五种武器
- 支持弹匣与备弹、换弹、开镜、后坐力、距离伤害衰减和武器掉落
- 服务器同步玩家位置、朝向、蹲伏状态、射击表现和生命值
- 服务端进行头部 / 身体命中检测；爆头伤害倍率为 4 倍
- HUD 显示生命值、护甲、弹药、回合比分、倒计时、FPS 和网络延迟
- 回合结束后双方恢复生命与武器状态，并返回各自出生点

离线模式提供训练 BOT：它会保持战斗距离、横向走位、检测视线并进行 3 至 6 发短连射，便于在没有第二名玩家或大厅服务器离线时测试武器和回合系统。

在线模式使用 Godot ENet 与专用大厅服务器。仓库包含大厅、房间、准备状态、玩家状态同步及服务端伤害判定代码；公开服务器是否可连接取决于专用服务器和公网隧道是否正在运行。

## 4K 音游系统

- 解析自定义 TXT 谱面和多难度配置
- 支持 Tap / Hold 音符与 `D / F / J / K` 四轨输入
- Critical、Perfect、Great、Good、Miss 多级判定
- Combo、ACC 与满分 1,000,000 的归一化计分
- 流速、Offset、音量等音游设置
- 选歌、游玩 HUD、暂停与结算流程
- 以音频播放时钟驱动音画同步

## 运行环境

- Godot 4.4 或更高版本
- 开发验证环境：Godot 4.7

克隆仓库后，用 Godot 导入根目录的 `project.godot`，然后运行主场景即可。

```bash
git clone https://github.com/CaptainLand/neon-coast-club.git
cd neon-coast-club
godot --editor project.godot
```

## 基本操作

| 操作 | 按键 |
| --- | --- |
| 移动 / 奔跑 | `WASD` / `Shift` |
| 跳跃 | `Space` |
| 交互、拾取 | `E` |
| 物品栏 | `1` - `5` / 鼠标滚轮 |
| 攻击、投掷 | 鼠标左键 / 右键 |
| 切换昼夜速度 | `T` |
| 自由飞行 | `F` |
| 获取或收起武器 | `B` |
| 换弹 | `R` |
| 暂停 | `Esc` |
| 全屏切换 | `F11` |
| 调试信息 | `F3` |

## 项目结构

```text
assets/       模型、贴图、音频、音游谱面与 UI 资源
scenes/       主场景、玩家和功能场景
scripts/      游戏、交互、网络、道具与音游逻辑
shaders/      海面、CRT、舞池、玻璃和视觉效果
docs/         设计与实现记录
project.godot Godot 项目入口
```
