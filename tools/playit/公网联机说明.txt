NEON COAST CLUB 公网联机（房主电脑）

首次设置：
1. 双击“启动公网隧道.cmd”。
2. Playit 会给出认领链接；在浏览器登录或注册 Playit，并认领这台电脑。
3. 在 Playit 管理页新建 Custom UDP 隧道。
4. Local address 填 127.0.0.1，Local port 填 27888。
5. 保存后记下 Playit 给出的公网地址，例如 example.playit.gg:12345。

每次开房：
1. 保持 playit.exe 运行。
2. 打开游戏，Esc -> 多人游戏，登录后端口保持 27888，点击“创建房间”。
3. 把 Playit 的“域名:端口”发给朋友。
4. 朋友登录游戏，把整段地址粘贴到“服务器 IP / 公网域名”输入框，点击“加入房间”。
5. 双方点击准备。

注意：
- 不要点击“互联网一键开房”，那个按钮走 UPnP；使用 Playit 时点普通“创建房间”。
- Windows 防火墙询问时允许游戏和 playit.exe 访问专用/公用网络。
- 房主必须同时保持游戏和 Playit 运行。
- 朋友不需要安装 Playit，也不需要设置端口转发。
